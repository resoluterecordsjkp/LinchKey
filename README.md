 # LinchKey

**Build:** Contest PWA v1.0.1

## Tagline

**Find What Governs.**

LinchKey is a source-first authority mapping and explainable reasoning system. It separates facts, governing authority, procedural requirements, conflicts, missing facts, and unresolved questions before presenting a conclusion.

Unlike traditional systems that often summarize or flatten complex legal and procedural questions, LinchKey follows the governing authority step by step and visibly shows why a conclusion is reached—or why one cannot yet be reached.

---

# Rule 0

**Never assume. Never flatten. Simplify the explanation—never the governing meaning.**

Unknown facts remain unknown.

One missing required element is not treated as satisfied simply because other elements are present.

---

# Core Principles

- Source-first reasoning
- Governing authority before conclusions
- Explain every conclusion
- Preserve uncertainty when facts are missing
- Never invent facts
- Never assume missing procedural steps occurred
- Always distinguish verified facts from assumptions
- Separate policy, procedure, opinion, and controlling authority
- Test every required legal element individually
- Recheck the reasoning path before presenting a result

# Important

LinchKey is designed to assist analysis by exposing governing authority, procedural gates, missing facts, and conflicts.

It does **not** invent facts, replace legal research, or substitute for professional judgment.

Instead, it provides a transparent reasoning path so users can understand **why** a conclusion is supported, unsupported, or unresolved.
